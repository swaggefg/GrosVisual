Documentation

Complete guide to installing and using the Protocol Minecraft mod

System Requirements

Minimum Requirements:

  • OS: Windows 10 (64-bit), macOS 10.14+, or Linux
  • Java: Java 8 or higher (Java 11+ recommended)
  • RAM: 4GB minimum
  • Storage: 5GB free disk space
  • GPU: Intel HD Graphics or equivalent

Recommended Requirements:

  • OS: Windows 10/11 (64-bit), macOS 11+, or Linux (latest kernel)
  • Java: Java 11 or Java 17
  • RAM: 8GB or more
  • Storage: 15GB SSD
  • GPU: NVIDIA GTX 1060 or AMD RX 580 or better

Installation Guide

Step 1: Install Java

First, ensure you have Java installed on your system. Download from java.com or use your package manager.

Step 2: Choose Your Mod Loader

Protocol supports multiple mod loaders. Choose one based on your preference:

  • Forge: Most compatible, largest mod ecosystem
  • Fabric: Lighter weight, faster loading times
  • Quilt: Modern alternative to Fabric

Step 3: Download Protocol

Download the latest version from the Downloads page. Select the correct version for your Minecraft version.

Step 4: Install the Mod

Place the downloaded Protocol JAR file into your mods folder:

# Windows %APPDATA%\.minecraft\mods # macOS ~/Library/Application Support/minecraft/mods # Linux ~/.minecraft/mods

Step 5: Launch Minecraft

Open the Minecraft launcher and select the appropriate Profile (Forge/Fabric/Quilt). Click Play and wait for the game to start.

Step 6: Verify Installation

Once in-game, press the Protocol key (default: Ctrl+P) to open the main menu. If you see the Protocol panel, the installation was successful!

Getting Started

First Launch

On your first launch, Protocol will create a configuration file and initialize the mod. This process takes about 30 seconds.

Main Menu

Press Ctrl+P (or your custom keybinding) to open the Protocol main menu. Here you can access:

  • Settings: Configure all mod options
  • Quests: Track your quest progress
  • Inventory: Enhanced inventory management
  • Multiplayer: Connect to servers
  • About: Information and help

Basic Commands

Protocol includes several useful commands. Use them in the chat:

/protocol help - Show all available commands /protocol config - Open configuration menu /protocol quest [id] - Start or check a quest /protocol teleport - Fast travel to locations /protocol stats - View your statistics

Keybindings

Customize keybindings in Settings > Controls. Default keybindings:

  • Ctrl+P: Open Protocol menu
  • Shift+Q: Open Quest log
  • Shift+I: Open enhanced inventory
  • Shift+M: Open map
  • Shift+K: Open skill tree

Configuration

Config File Location

# Windows %APPDATA%\.minecraft\config\protocol\settings.json # macOS ~/Library/Application Support/minecraft/config/protocol/settings.json # Linux ~/.minecraft/config/protocol/settings.json

Common Configuration Options

{ "protocol": { "enabled": true, "language": "en", "theme": "dark", "performance": { "max_fps": 144, "render_distance": 16, "particle_level": "high" }, "gameplay": { "difficulty_scaling": true, "quest_notifications": true, "auto_save_interval": 300 }, "ui": { "scale": 1.0, "transparency": 0.8, "show_coordinates": true } } }

Resetting Configuration

To reset Protocol to default settings, delete the config folder and restart Minecraft. The configuration will be regenerated automatically.

Troubleshooting

Protocol won't load

  • Check Java version: java -version
  • Verify mod file is in correct mods folder
  • Delete config folder and restart
  • Check logs: .minecraft/logs/latest.log

Game crashes on startup

  • Increase allocated RAM in launcher
  • Check for mod conflicts
  • Update to latest Java version
  • Report crash with log file to support

Low performance / FPS drops

  • Lower render distance in settings
  • Reduce particle level
  • Disable visual enhancements if needed
  • Check if GPU drivers are up to date
  • Allocate more RAM to Minecraft

Features not working

  • Check if mod is properly installed
  • Verify keybindings in Settings
  • Update to latest Protocol version
  • Check mod compatibility with other mods

Multiplayer issues

  • Protocol must be installed on server
  • Use compatible server version
  • Check firewall settings
  • Verify network connection

Updating Protocol

Automatic Updates

Protocol can notify you of new updates. Enable automatic checks in Settings > Updates.

Manual Update

To manually update:

  1. Download the new version from Downloads page
  2. Delete or backup the old Protocol JAR file
  3. Place the new JAR file in your mods folder
  4. Restart Minecraft

Update Notes

Always check the changelog before updating to understand what has changed. Major version updates may require config reset.

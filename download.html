Quick Start Installation

Step-by-Step Guide:

  1. Download Protocol: Click the download button above
  2. Install Java: Make sure Java 8+ is installed
  3. Choose a mod loader: Download and install Forge, Fabric, or Quilt
  4. Locate your mods folder:
    Windows: %APPDATA%\.minecraft\mods
    macOS: ~/Library/Application Support/minecraft/mods
    Linux: ~/.minecraft/mods
  5. Copy the JAR file: Place Protocol JAR in your mods folder
  6. Launch Minecraft: Select the correct profile and play!
  7. Verify Installation: Press Ctrl+P in-game to open Protocol

Need help? Check out our full installation guide or ask in our Discord community.
